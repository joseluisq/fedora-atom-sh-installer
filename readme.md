# Atom Shell Installer
> Small [Atom](https://atom.io/) shell script installer for [Fedora Linux x86_x64](https://getfedora.org/en/workstation/)

## Install

```sh
$ wget -qO- http://git.io/fed-atom-sh | sudo sh
```

## Usage

```sh
$ atom my-directory-or-file-path
```
